==Switch-Case==

It is analogue of *if-else* statements:

```cs
string myCow = "cow";
string myCapCow = "Cow";

switch(myCow)
{
	case "cow":
		Console.WriteLine("Lowercase");
		break; // break is obligatory
	case "Cow":
		Console.WriteLine("Uppercase");
		break;
	default:
		Console.WriteLine("Default");
		break;
}
```
